# 圖片壓縮與優化指南

## 📋 目標
將 `圖片封存` 資料夾的圖片壓縮成 WebP 格式，放到 `img` 資料夾，加快網站載入速度。

---

## 🎯 方法一：線上工具（最簡單）

### 推薦工具：Squoosh.app
1. 打開 https://squoosh.app/
2. 拖曳圖片到網頁
3. 右側選擇：
   - **格式**：WebP
   - **品質**：70-80
   - **尺寸**：最長邊 1200px（設備圖可以 800px）
4. 點選「下載」
5. 將檔案重新命名並放到 `img` 資料夾

## 🎯 方法二：批次壓縮（進階）

### 使用 ImageMagick（需安裝）

#### 安裝 ImageMagick
1. 下載：https://imagemagick.org/script/download.php#windows
2. 安裝時勾選「Add to PATH」

#### 批次轉換指令
在 PowerShell 執行：

```powershell
# 進入圖片資料夾
cd "C:\Users\張-1\CascadeProjects\kenwholesale\圖片封存"

# 批次轉換所有 JPG/PNG 為 WebP（品質 75，最長邊 1200px）
Get-ChildItem -Filter *.jpg | ForEach-Object {
    $output = "..\img\$($_.BaseName).webp"
    magick convert $_.FullName -resize 1200x1200^ -quality 75 $output
    Write-Host "✅ 已轉換: $($_.Name) → $output"
}

Get-ChildItem -Filter *.png | ForEach-Object {
    $output = "..\img\$($_.BaseName).webp"
    magick convert $_.FullName -resize 1200x1200^ -quality 75 $output
    Write-Host "✅ 已轉換: $($_.Name) → $output"
}
```

## 🎯 方法三：使用 XnConvert（免費軟體）

1. 下載：https://www.xnview.com/en/xnconvert/
2. 安裝後打開
3. **輸入**：拖曳 `圖片封存` 資料夾所有圖片
4. **動作**：
   - 新增動作 → 影像 → 調整大小
   - 最長邊：1200 像素
   - 保持比例：勾選
5. **輸出**：
   - 格式：WebP
   - 品質：75
   - 資料夾：選擇 `img` 資料夾
6. 點選「轉換」

---

## 📝 轉換後的步驟

### 1. 重新命名檔案
將檔名改成對應的商品 ID（參考上面的對應表）

### 2. 更新試算表「圖片網址」欄位
填入相對路徑，例如：
```
img/powder_case.webp
img/waterchest_case.webp
img/milkfried_case.webp
```

### 3. 上傳到 GitHub
1. 將整個 `img` 資料夾上傳到 GitHub
2. 同時上傳更新後的 `index.html`

### 4. 測試
打開網站，圖片應該會快速載入且清晰！

## 🆘 如果遇到問題

1. **圖片模糊**：提高品質到 80-85
2. **檔案太大**：降低尺寸到 800px
3. **格式不支援**：改用 JPG（品質 80）
