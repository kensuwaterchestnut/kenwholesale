### 3️⃣ 修改 index.html（15分鐘）

**找到 index.html 的第 370 行附近：**

```javascript
/* ===== 預設商品 ===== */
const DEFAULT_PRODUCTS = [
  // ... 一大堆商品資料
];
```

**改成：**

```javascript
/* ===== 商品資料（從 API 載入） ===== */
let products = [];  // 不再寫死，從 API 載入

// 載入商品資料
async function loadProductsFromAPI() {
  try {
    const response = await fetch(GAS_ENDPOINT);
    const data = await response.json();
    
    if (data.products && Array.isArray(data.products)) {
      // 合併圖片 URL
      products = data.products.map(p => {
        let imgUrl = '';
        
        // 從圖片資料中尋找對應的圖片
        for (const images of Object.values(data.images || {})) {
          const matched = images.find(img => 
            img.name === p.imgFilename || 
            img.name.replace(/\.[^.]+$/, '') === p.id
          );
          if (matched) {
            imgUrl = matched.url + '&v=' + matched.ts;
            break;
          }
        }
        
        return { ...p, img: imgUrl };
      });
      
      console.log('✅ 載入 ' + products.length + ' 個商品');
      return true;
    }
  } catch (error) {
    console.error('❌ 載入失敗:', error);
    return false;
  }
}
```

**修改 loadProducts() 函數（如果有的話）：**

```javascript
function loadProducts(){
  // 刪除或註解掉原本的內容
  // 改用從 API 載入的 products
  return products;
}
```

**修改頁面載入邏輯（在最底部找到）：**

```javascript
// 原本可能是這樣：
applyBrand();
renderChips();
renderGrid();
updateCartCount();

// 改成：
window.addEventListener('DOMContentLoaded', async function() {
  // 1. 先載入商品
  const loaded = await loadProductsFromAPI();
  
  if (!loaded) {
    alert('商品資料載入失敗，請檢查 API 設定');
    return;
  }
  
  // 2. 然後渲染頁面
  applyBrand();
  renderChips();
  renderGrid();
  updateCartCount();
});
```

## 🎯 常見問題

### Q1: 為什麼還要填 FOLDER_MAP？
**A:** 圖片還是放在 Drive，需要指定資料夾位置。商品資料在 Sheet，圖片在 Drive。

### Q2: 可以不用 Drive 嗎？
**A:** 可以，但圖片 URL 要填在 Sheet 的「圖片檔名」欄位（填完整 URL）。

### Q3: Sheet 改了，網頁沒變？
**A:** 
1. 確認 GAS 有重新部署
2. 清除瀏覽器快取（Ctrl+Shift+Delete）
3. 檢查 F12 Console 有無錯誤

### Q4: 可以匯出備份嗎？
**A:** 可以！檔案 → 下載 → Excel (.xlsx) 或 CSV
