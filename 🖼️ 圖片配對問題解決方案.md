# 🖼️ 圖片配對問題解決方案

## 🔍 問題分析

您遇到的問題是：**試算表中填寫了商品 ID 和圖片檔名，但前端無法顯示雲端資料夾的圖片**。

---

## 📋 目前的配對邏輯

### 試算表欄位結構（第 15 欄 = `imgFilename`）
```
A欄: ID
B欄: 標籤
C欄: 品名
D欄: 類型
E欄: 規格
...
O欄: imgFilename（圖片檔名）← 第 15 欄
```

### 前端配對邏輯（`matchProductImage` 函數）
```javascript
function matchProductImage(product, images) {
  const filename = product.imgFilename || '';
  
  // 搜尋所有溫層的圖片
  const allImages = [
    ...(images.常溫 || []),
    ...(images.冷藏 || []),
    ...(images.冷凍 || [])
  ];
  
  // 精確配對檔名（去除副檔名）
  for (const img of allImages) {
    if (img.name.includes(filename.replace('.jpg', '').replace('.png', ''))) {
      return img.url;
    }
    if (img.name.includes(product.id)) {
      return img.url;
    }
  }
  
  return '';
}
```

## ✅ 正確的使用方式

### 方法 1：使用圖片檔名配對（推薦）

**試算表設定：**
| ID | 品名 | ... | imgFilename |
|----|------|-----|-------------|
| P001 | 芋泥丁 | ... | `芋泥丁` |
| P002 | 菱角酥 | ... | `菱角酥` |

**雲端資料夾檔名：**
- `芋泥丁.jpg` ✅
- `菱角酥.png` ✅
- `芋泥丁-產品照.jpg` ✅（也能配對）

**重點：**
- ✅ `imgFilename` 欄位填寫**不含副檔名**的檔名（例如：`芋泥丁`）
- ✅ 雲端資料夾的檔名要**包含**這個名稱（例如：`芋泥丁.jpg`）
- ✅ 支援中文、英文、數字檔名
- ✅ 副檔名可以是 `.jpg`、`.png`、`.webp` 等

---

### 方法 2：使用商品 ID 配對

**試算表設定：**
| ID | 品名 | ... | imgFilename |
|----|------|-----|-------------|
| P001 | 芋泥丁 | ... | （留空） |
| P002 | 菱角酥 | ... | （留空） |

**雲端資料夾檔名：**
- `P001.jpg` ✅
- `P002.png` ✅
- `P001-芋泥丁.jpg` ✅（也能配對）

**重點：**
- ✅ 如果 `imgFilename` 留空，會自動用 `ID` 配對
- ✅ 檔名要**包含**商品 ID

## 🔧 常見問題排查

### ❌ 問題 1：填寫了檔名但沒有圖片

**可能原因：**
1. `imgFilename` 欄位填寫了**完整檔名含副檔名**（例如：`芋泥丁.jpg`）
2. 雲端資料夾的檔名**不包含**這個名稱

**解決方法：**
```
錯誤：imgFilename = "芋泥丁.jpg"  ❌
正確：imgFilename = "芋泥丁"      ✅

雲端檔名：芋泥丁.jpg ✅
```

---

### ❌ 問題 2：圖片在錯誤的資料夾

**檢查：**
- 常溫商品的圖片要放在「常溫」資料夾（ID: `14X6WbIQd6mAiLOEE_h73mkXdQL1gu0mM`）
- 冷藏商品的圖片要放在「冷藏」資料夾（ID: `1vo3Zbwt008r684mAm9nakghTO2zojWmf`）
- 冷凍商品的圖片要放在「冷凍」資料夾（ID: `1IETjj7n9nlQGsrCho_zxF81p4OWLCvWZ`）

**注意：** 前端會搜尋**所有溫層資料夾**，所以放錯資料夾也能找到，但建議按溫層分類。

---

### ❌ 問題 3：修改後沒有更新

**解決步驟：**
1. 修改試算表後，等待 **30 秒**（自動更新）
2. 或者手動點選「啃酥面板」→「🚀 手動更新商品到前端」
3. 重新整理前端頁面（Ctrl+F5 強制重新整理）

## 🚀 快速修復步驟

1. **檢查試算表 O 欄（imgFilename）**
   - 確認填寫的是**不含副檔名**的檔名
   - 例如：`芋泥丁` 而不是 `芋泥丁.jpg`

2. **檢查雲端資料夾的檔名**
   - 檔名要**包含** imgFilename 的內容
   - 例如：`芋泥丁.jpg`、`芋泥丁-產品照.png` 都可以

3. **手動更新商品版本**
   - 試算表 → 啃酥面板 → 🚀 手動更新商品到前端

4. **重新整理前端**
   - 按 Ctrl+F5 強制重新整理

5. **檢查 Console**
   - 按 F12 查看是否有錯誤訊息
