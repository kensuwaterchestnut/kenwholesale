# 🔍 圖片顯示問題診斷

## 問題描述
測試工具顯示圖片配對成功，但前端網頁無法顯示圖片。

---

## 可能的原因

### 1️⃣ localStorage 快取問題 ⭐ 最常見
**症狀**：測試工具顯示有圖片，但網頁沒有
**原因**：瀏覽器使用了舊的 localStorage 資料（沒有圖片網址）

**解決方法**：
1. 按 `F12` 打開開發者工具
2. 切換到「Console」分頁
3. 輸入以下指令並按 Enter：
   ```javascript
   localStorage.clear()
   ```
4. 重新整理頁面（`Ctrl+F5`）

---

### 2️⃣ GAS 網址不一致
**症狀**：前端使用舊的 GAS 網址
**原因**：部署後沒有更新前端的 `GAS_ENDPOINT`

**檢查方法**：
1. 打開網頁
2. 按 `F12` → Console
3. 查看是否有「🌐 從 GAS 載入商品資料...」
4. 查看是否有「✅ 載入 XX 個商品」

**已修正**：我已經更新為您提供的新網址
```
https://script.google.com/macros/s/AKfycbx0gEN3TdKb5PI36hpav9YKe0HFDGehEPUJmK7IwEJNdRwn4WV5fHvnmDQBFrN_V-ivQQ/exec
```

---

### 3️⃣ 圖片配對邏輯問題
**症狀**：Console 顯示「❌ 無法配對」
**原因**：試算表的 `imgFilename` 與雲端檔名不匹配

**檢查方法**：
1. 按 `F12` → Console
2. 查看配對訊息：
   ```
   🔍 配對商品: 啃酥專用粉 (啃酥專用粉), 可用圖片: 6 張
   ✅ 完全匹配: 啃酥專用粉 → 啃酥專用粉.jpg
   ```

**已修正**：我已經加入詳細的除錯訊息

---

### 4️⃣ 圖片網址失效
**症狀**：Console 沒有錯誤，但圖片顯示為破圖
**原因**：Google Drive 圖片權限設定錯誤

**檢查方法**：
1. 按 `F12` → Network 分頁
2. 重新整理頁面
3. 找到圖片請求（`.jpg` 結尾）
4. 查看 Status Code：
   - `200`：正常 ✅
   - `403`：權限錯誤 ❌
   - `404`：檔案不存在 ❌

**解決方法**：
確認 Google Drive 資料夾的共用設定：
- 常溫資料夾：`14X6WbIQd6mAiLOEE_h73mkXdQL1gu0mM`
- 冷藏資料夾：`1vo3Zbwt008r684mAm9nakghTO2zojWmf`
- 冷凍資料夾：`1IETjj7n9nlQGsrCho_zxF81p4OWLCvWZ`

每個資料夾都要設定為「知道連結的任何人」可以檢視。

---

### 5️⃣ 圖片還沒載入完成
**症狀**：頁面剛打開時沒圖片，幾秒後才出現
**原因**：GAS API 回應較慢

**正常行為**：
1. 頁面載入 → 顯示 `DEFAULT_PRODUCTS`（舊圖片或無圖）
2. GAS 載入完成 → 更新為新圖片

**不需要修正**，這是正常的載入流程。

## 💡 快速修復

如果上述步驟都檢查過了，還是沒圖片，請執行：

```javascript
// 在 Console 中執行這段完整的診斷程式碼
(async () => {
  console.log('=== 開始診斷 ===');
  
  // 1. 清除快取
  localStorage.clear();
  console.log('✅ 已清除 localStorage');
  
  // 2. 重新載入資料
  const response = await fetch('https://script.google.com/macros/s/AKfycbx0gEN3TdKb5PI36hpav9YKe0HFDGehEPUJmK7IwEJNdRwn4WV5fHvnmDQBFrN_V-ivQQ/exec');
  const data = await response.json();
  console.log('✅ GAS 回應:', data);
  
  // 3. 檢查圖片
  if (data.images) {
    const total = (data.images.常溫?.length || 0) + (data.images.冷藏?.length || 0) + (data.images.冷凍?.length || 0);
    console.log(`✅ 圖片總數: ${total} 張`);
    console.log('常溫:', data.images.常溫);
    console.log('冷藏:', data.images.冷藏);
    console.log('冷凍:', data.images.冷凍);
  }
  
  // 4. 檢查商品
  if (data.products) {
    console.log(`✅ 商品總數: ${data.products.length} 個`);
    console.log('第一個商品:', data.products[0]);
  }
  
  console.log('=== 診斷完成，請重新整理頁面 ===');
  location.reload();
})();
```
